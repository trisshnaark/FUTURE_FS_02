## 🚀 How to Start Using Your CRM

### Option 1: Quick Start (5 minutes)
```bash
# 1. Setup Backend
cd backend
cp config/.env.example .env
npm install
npm start

# 2. Setup Frontend (new terminal)
cd frontend
npm install
npm run dev

# 3. Create Database
mysql -u root -p
CREATE DATABASE crm;
USE crm;
SOURCE ../backend/database.sql;

# 4. Access at http://localhost:5173
# Login: admin@crm.com / admin123
```

Follow SETUP_GUIDE.md step-by-step for comprehensive instructions.
